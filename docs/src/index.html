PRPL

HTML-based static site generator implementing the PRPL pattern.

Start developing in one command:

npm init prpl

or head over to the Getting started guide.

Why PRPL?

🚰 No kitchen sink

Chances are you have tried other other static site generators and thought they were overkill. PPRL can be used just to interpolate your content into HTML.

🎛 No bundlers, configuration or underlying frameworks

The worst kind of technical debt comes from tooling and dependency trees. PRPL stays small and gets out of your way.

🚪 Easy to remove

The second doc after Getting started is Removing PRPL. PRPL shows you the door before you invest.

⚡️ Very, very fast

Using the namesake pattern this project is built with, PRPL uses modern prefetching and routing systems to help your site achieve max speed.
